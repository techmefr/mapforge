# Tilesets

Layer fonctionnel de generation de tuiles de terrain. Porte les
definitions de terrains (couleurs, motifs), le rendu procedural sur
`<canvas>`, l'analyse du prompt texte en mots-cles de terrain, et
l'editeur de tileset (forme, taille, variantes, generation, suppression).

S'appuie sur le store du layer `Games` pour lire/modifier le tileset
selectionne.
